## üöÄ Features

* **No Root Required**: Works on any standard Android device.
* **High Performance**: Uses the native Go implementation of DNSTT for maximum speed.
* **Simple Interface**: One-tap connection.
* **Secure**: Tunnel your traffic securely over DNS.
